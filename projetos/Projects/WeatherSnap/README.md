# 🌤️ WeatherSnap

> Dashboard responsivo de previsão do tempo integrado a uma API meteorológica, com temas dinâmicos e dicas contextuais.

## 📋 Sobre o Projeto

WeatherSnap é uma aplicação web de previsão do tempo. Ela consome dados de uma API meteorológica externa para apresentar informações atuais e a previsão para os próximos 3 dias de forma elegante e intuitiva. 

## ✨ Funcionalidades

- **Busca de Cidades:** Permite buscar o clima de qualquer cidade do mundo.
- **Theming Dinâmico:** O visual do dashboard muda de acordo com o clima atual (ensolarado, chuvoso, nublado, etc.).
- **Dicas Contextuais:** Exibe dicas inteligentes com base no clima.
- **Design Responsivo:** Layout adaptado para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura e semântica.
- **TailwindCSS:** Framework utility-first para estilização rápida e responsiva.
- **JavaScript (ES6+):** Lógica da aplicação, manipulação do DOM e requisições assíncronas.
- **Fetch API:** Consumo da API de previsão do tempo.

## 🚀 Como Executar

Abra o arquivo `index.html` em seu navegador para visualizar a aplicação em funcionamento. Para uma melhor experiência durante o desenvolvimento, recomenda-se o uso do Live Server.

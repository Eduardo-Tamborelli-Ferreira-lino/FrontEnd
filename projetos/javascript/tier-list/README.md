# 🎮 Tier List — Drag & Drop

> Aplicação interativa de Tier List utilizando a API nativa de Drag & Drop do HTML5.

## 📋 Sobre o Projeto

Uma ferramenta web interativa para classificar itens (neste exemplo, campeões do jogo League of Legends) em diferentes níveis (S, A, B, C, D). O projeto foca no uso avançado do DOM e eventos de arrastar e soltar (`drag` e `drop`).

## ✨ Funcionalidades

- Interface intuitiva de arrastar e soltar.
- Múltiplos Tiers disponíveis para ranqueamento.
- Mais de 150 imagens de personagens para ranquear.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura e API nativa de Drag & Drop (`draggable="true"`, eventos `ondragstart`, `ondragover`, `ondrop`).
- **CSS3:** Estilização visual dos tiers e grid de seleção.
- **JavaScript (ES6+):** Lógica que gerencia o estado dos itens sendo arrastados e atualiza a interface.

## 🚀 Como Executar

Abra o arquivo `index.html` em seu navegador web para iniciar o ranqueamento.

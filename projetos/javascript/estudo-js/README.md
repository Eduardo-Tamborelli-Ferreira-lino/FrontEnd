# 📦 Estudos Gerais de JavaScript

> Estudos aprofundados sobre recursos modernos do JavaScript, incluindo modularização e exercícios lógicos.

## 📋 Sobre os Estudos

Este diretório contém aprofundamentos nas regras de negócio e na arquitetura do JavaScript.

## ✨ Conteúdo

- **Lista de Exercícios JS:** Práticas de lógica de programação, algoritmos, loops e condicionais utilizando JS.
- **Modularização (ES6):** Estudos focados no sistema de módulos nativo do JavaScript moderno.
  - Uso de `import` e `export`.
  - Separação de responsabilidades.
  - Boas práticas de arquitetura Front-end.

## 🛠️ Tecnologias Utilizadas

- **JavaScript (ES6+)**

## 🚀 Como Executar

Para os exercícios lógicos, você pode executar os arquivos JS no Node.js via terminal ou abrir o console do navegador se estiverem vinculados a um arquivo HTML.
Para testar os módulos ES6, **é obrigatório** rodar as páginas HTML usando um servidor local (como o Live Server), pois navegadores bloqueiam `type="module"` utilizando o protocolo `file://`.

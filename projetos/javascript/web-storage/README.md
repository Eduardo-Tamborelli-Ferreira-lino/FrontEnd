# 💾 Web Storage — Gerenciador de Dados

> Aplicação que demonstra o uso de Web Storage (`localStorage`) para persistência de dados no browser.

## 📋 Sobre o Projeto

Este projeto demonstra como salvar informações diretamente no navegador do usuário, permitindo que os dados não sejam perdidos ao recarregar a página ou fechar o navegador.

## ✨ Funcionalidades

- **CRUD Básico:** Adicionar e remover usuários/dados da lista.
- **Persistência de Dados:** Uso de `localStorage` para armazenar a lista no formato JSON (stringify e parse).
- **Contador de Visitas:** Demonstra a persistência gravando quantas vezes a página foi acessada.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Interface simples para input e visualização dos dados.
- **JavaScript:** Manipulação do DOM e uso extensivo das Web APIs `localStorage.setItem()`, `localStorage.getItem()`, e `localStorage.removeItem()`.

## 🚀 Como Executar

1. Abra o arquivo `index.html` no seu navegador.
2. Adicione alguns dados via interface.
3. Recarregue a página (F5) ou feche e abra novamente.
4. Você observará que os dados continuam lá!

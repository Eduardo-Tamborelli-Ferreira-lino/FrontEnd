# 🧩 Manipulação do DOM

> Atividades práticas de manipulação do DOM (Document Object Model) com JavaScript puro.

## 📋 Sobre os Projetos

Esta pasta engloba exercícios fundamentais para entender como o JavaScript interage com o HTML, alterando conteúdos, estilos e respondendo a interações do usuário.

## ✨ Atividades Inclusas

- **Atividade 01:** Focada na seleção de elementos (`querySelector`, `getElementById`), alteração de propriedades e eventos básicos (cliques, inputs).
- **Atividade 02:** Focada em manipulação estrutural avançada (CRUD dinâmico), incluindo a criação (`createElement`) e remoção (`remove`) de elementos do DOM programaticamente.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Interfaces base para as atividades.
- **JavaScript (Vanilla):** Web APIs do DOM.

## 🚀 Como Executar

Acesse as pastas de cada atividade e abra o arquivo `index.html` em seu navegador.

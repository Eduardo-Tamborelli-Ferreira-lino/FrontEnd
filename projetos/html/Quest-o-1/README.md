# 🌍 Quest-o-1 — Site de Turismo

> Site de turismo com múltiplos destinos internacionais (Áustria, Japão, Suíça).

## 📋 Sobre o Projeto

Um site focado em turismo internacional, apresentando informações, imagens e atrações para diferentes países. O projeto serve como um estudo prático na criação de uma estrutura web multi-página.

## ✨ Destaques

- **Estrutura multi-página:** Navegação entre a página principal e as páginas de cada destino.
- **Destinos incluídos:** 
  - 🇦🇹 Áustria (Viena, Salzburg, Alpes)
  - 🇯🇵 Japão (Tóquio, Quioto, Monte Fuji)
  - 🇨🇭 Suíça (Zermatt, Lucerna, Interlaken)
- **Galeria de imagens:** Rica em fotografias dos locais.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Navegação semântica entre páginas.
- **CSS3:** Estilização geral e layout.

## 🚀 Como Executar

Para começar a navegar, abra o arquivo `index.html` raiz da pasta.

# 🖋️ HTML Project — Fontes Customizadas

> Projeto focado na exploração de layouts com fontes customizadas.

## 📋 Sobre o Projeto

Um estudo prático envolvendo a importação e o uso de fontes não padronizadas na web (como Google Fonts ou `@font-face`), focado em aprimorar a tipografia e o design das páginas.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** (Tipografia e Web Fonts)

## 🚀 Como Executar

Abra o arquivo `index.html` em seu navegador para visualizar as fontes customizadas renderizadas na página.

# 🎨 Frameworks Frontend

> Estudos práticos dos principais frameworks de CSS: Bootstrap e Tailwind CSS.

## 📋 Sobre os Estudos

Esta pasta contém exercícios e pequenos projetos focados no aprendizado de ferramentas que aceleram o desenvolvimento Front-end.

## 🛠️ Frameworks Estudados

### Bootstrap
- Uso de componentes pré-estilizados.
- Compreensão do Grid System.
- Uso de classes utilitárias.

### Tailwind CSS
- Abordagem *utility-first*.
- Customização do tema (`tailwind.config.js`).
- Build pipeline.

## 🚀 Como Executar

As pastas internas (`BootStrap/` e `Tailwind/`) contêm seus respectivos arquivos `index.html`. 
Para o Tailwind, dependendo da configuração de build, pode ser necessário rodar o compilador via Node.js ou usar o CDN (conforme implementado nos exercícios).

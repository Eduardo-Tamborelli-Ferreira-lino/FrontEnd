# 💼 TalentHive

> Landing page profissional para plataforma de recrutamento e talentos.

## 📋 Sobre o Projeto

A TalentHive é uma landing page voltada para o mercado corporativo, focada em conectar empresas aos melhores talentos. O projeto demonstra a criação de seções típicas de páginas de conversão e marketing.

## ✨ Destaques

- **Seção Hero impactante:** Composta por chamadas de ação fortes.
- **Vídeo de introdução:** Integração de mídia de vídeo.
- **FAQ Interativo:** Seção de perguntas frequentes estruturada.
- **Mobile-first:** Layout totalmente responsivo, desenhado primariamente para dispositivos móveis e adaptado para telas maiores.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** (Flexbox/Grid e Media Queries)

## 🚀 Como Executar

Abra o arquivo `index.html` em qualquer navegador. Recomenda-se utilizar as ferramentas de desenvolvedor do navegador (F12) para visualizar a responsividade.

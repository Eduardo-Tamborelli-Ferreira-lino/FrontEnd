# 🍽️ Kristian Sabor & Arte

> Landing page para restaurante com design moderno e apresentação de pratos.

## 📋 Sobre o Projeto

Uma página web elegante criada para o fictício restaurante "Kristian Sabor & Arte". O site serve como um portfólio para apresentar os pratos, o ambiente do restaurante e incluir uma seção de avaliações dos clientes.

## ✨ Funcionalidades

- Design moderno e atrativo.
- Layout responsivo (adapta-se a diferentes tamanhos de tela).
- Seção de avaliações com estrelas visuais.
- Galeria de imagens dos pratos.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Marcação semântica.
- **CSS3:** Estilização com Flexbox/Grid, transições e media queries.

## 🚀 Como Executar

Execute abrindo o arquivo `index.html` no seu navegador favorito.

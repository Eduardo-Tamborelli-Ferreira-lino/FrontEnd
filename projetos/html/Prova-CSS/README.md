# 📊 Prova CSS — Dashboard Analytics

> Prova prática que reproduz um dashboard de analytics utilizando apenas HTML e CSS puro.

## 📋 Sobre o Projeto

Este projeto foi desenvolvido como uma avaliação prática para testar habilidades avançadas de CSS. O objetivo principal foi recriar um layout de dashboard analítico do zero, focando na organização em grid, posicionamento de elementos e responsividade sem o uso de frameworks externos.

## ✨ Destaques

- Layout de dashboard completo.
- Sidebar de navegação fixa.
- Cards de métricas alinhados.
- Gráficos puramente visuais usando técnicas de CSS.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3** (Flexbox, CSS Grid, variáveis CSS)

## 🚀 Como Executar

Para visualizar o projeto, abra o arquivo `index.html` no navegador.

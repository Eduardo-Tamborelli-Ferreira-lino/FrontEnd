# 📚 Estudo Recuperação

> Estudos de recuperação e revisão focados em fundamentos web.

## 📋 Sobre o Projeto

Uma pasta dedicada à revisão e fixação de conceitos básicos e intermediários de desenvolvimento web. O foco é resolver problemas e construir pequenos layouts para solidificar o aprendizado.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

## 🚀 Como Executar

Você pode encontrar arquivos `.html` individuais dentro desta pasta. Abra qualquer um deles em seu navegador para visualizar as páginas.

# ⏳ Cápsula do Tempo

> Projeto voltado para registrar mensagens ou objetivos para o futuro.

## 📋 Sobre o Projeto

Este projeto consiste em uma página web que funciona como uma "Cápsula do Tempo", onde os usuários podem registrar mensagens e objetivos para o seu "eu" do futuro.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização visual e responsividade.

## 🚀 Como Executar

Para visualizar o projeto, basta abrir o arquivo principal (`index.html`) diretamente no seu navegador, ou utilizar a extensão Live Server do VSCode.
